# 📝 Gerenciador de Tarefas Simples (Task Manager API)

## 📌 Sobre o Projeto

Este projeto é uma **Web API REST** desenvolvida em **C#** e **.NET** como parte do desafio prático da trilha de aprendizado da Rocketseat.

O objetivo principal é criar uma API completa para gerenciamento de tarefas (CRUD - Create, Read, Update, Delete), aplicando boas práticas de arquitetura em camadas, separando a **Camada de Comunicação** (Controllers) da **Camada de Regras de Negócio** (Services).

### Funcionalidades

A API permite realizar as seguintes operações:
* **Criar** uma nova tarefa (POST).
* **Listar** todas as tarefas existentes (GET).
* **Buscar** uma tarefa específica por ID (GET).
* **Atualizar** informações de uma tarefa (PUT).
* **Excluir** uma tarefa por ID (DELETE).
